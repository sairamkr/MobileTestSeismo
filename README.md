# Speculative-Seismograph

Speculations by Sem06 2026 Interaction Design Students.
An interactive artifact that displays speculative fictions on a seismograph drum. 

## How it works

The five switches form a 5-bit binary string that becomes a filename. This gives 32 possible
combinations (`00000`–`11111`), and a pair of story files exists for each. Flipping any switch
resets the scroll and loads one of the two matching stories.
